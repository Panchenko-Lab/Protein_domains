This directory contains all files related to the paper.

### 01_protein_sequences:

Contains the protein sequences 

### 02_assigned_domains:

Includes the domain annotations assigned to each protein sequence.

### 03_aligned_domains:

Contains multiple sequence alignments (MSAs) of protein domains.

### 04_mapped_mutations:

Includes mutation data mapped onto the aligned protein domains.

### 05_benchmark 

The dataset containing mutations and labels.
